# Project Vision: The Secret of Moo Yang
**Genre:** 2D Point-and-Click Adventure (Mobile Web)
**Tech Stack:** Vue 3 + Vite + Tailwind CSS
**Target Platform:** Mobile Browsers (iOS Safari / Android Chrome)
**Current PI Objective:** Playable Prologue PoC (Hotel to BBQ Stall).

## Interaction Model (Modern Contextual)
* **Left-Click / Single Tap:** Contextual Action. If clicking an NPC, "Talk". If an item, "Pick Up". If a door, "Open".
* **Right-Click / Long Press (minimum 500ms):** Examine. Provides a humorous, descriptive observation from Hans's perspective.
* **UI:** Must be touch-friendly with large hit-boxes for mobile users.

## PI 1: The Prologue Goal
Implement the opening sequence: 
1. **Scene 1:** Hans at a 5-star hotel in Pattaya (The Recap).
2. **Scene 2:** Transition to the "Moo Yang Miner 3000" stall.
3. **The Puzzle:** Successfully initialize the miners using the greasy USB stick found in Scene 2.

## Deployment
* Every successful Sprint must build and deploy to GitHub Pages.
