# 💎 The Myth of Crystal Island
[![License: CC BY-NC-ND 4.0](https://img.shields.io/badge/License-CC%20BY--NC--ND%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-nd/4.0/)

> "Efficiency is not an option; it is a requirement... even when mining pork." — Hans

**The Myth of Crystal Island** is an autonomous, mobile-first point-and-click adventure game. It follows the journey of **Hans**, a rigid Industrial Engineer from Stuttgart, who inherits a "thermal energy enterprise" in Pattaya, Thailand, only to discover it's a roadside BBQ stall powered by the heat of Bitcoin miners.

---

## 🛠 Tech Stack
* **Framework:** Vue 3 (Vite)
* **Styling:** Tailwind CSS
* **State Management:** Pinia
* **Interactions:** `@vueuse/core` (Long-press support)
* **Orchestration:** Python-based Agile Driver + OpenClaw Agent

---

## 🕹 Game Mechanics (Modern Contextual)
This game is optimized for Mobile Web (iOS/Android) and Desktop:
* **Single Tap / Left Click:** Contextual Action (Talk, Pick Up, Use).
* **Long Press / Right Click:** Examine (Hans provides a precise, often condescending engineering analysis).

---

## 🤖 The Autonomous Factory
This project is developed using an **Agile Autonomous Loop**. An AI agent (OpenClaw) manages the codebase 24/7 through a series of "Sprints."

### The Agile Loop
1.  **Planning:** Agent analyzes `goal.md` and `story.md`.
2.  **Implementation:** Agent writes Vue components and game logic.
3.  **Verification:** Automated `npm run build` and `npm test`.
4.  **Deployment:** Successful builds are automatically pushed to GitHub.
5.  **Review:** A `sprint_x_review.md` is generated to document progress.

### Planning Intervals (PI)
The project operates in **Planning Intervals**. While the agent runs infinitely, a human "Product Owner" can trigger a PI Planning session by creating `START_PI_PLANNING.txt` to update the vision or expand the agent's skills.

---

## 📊 Development Dashboard
The current state of the "Factory" is tracked in `dashboard.json`. 
* **System Status:** Check here to see if the agent is currently coding, verifying, or resting.
* **Sprint ID:** View the current progress through the story.

## 📁 Project Structure
* `goal.md`: The strategic North Star.
* `skills.md`: The agent's authorized tools and rules.
* `story.md`: The narrative script and dialogue.
* `theme.md`: The visual and tonal art direction.
* `driver.py`: The Python orchestrator running the loop.
